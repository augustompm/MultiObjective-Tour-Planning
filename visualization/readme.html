Visualizador de Resultados NSGA-II

1Instalação de Dependências

Antes de iniciar a visualização, certifique-se de instalar todas as dependências necessárias:

pip install -r requirements.txt

2Iniciar o Visualizador

Execute o script Python para iniciar o servidor de visualização:

python results_visualizer.py

O servidor será iniciado e o dashboard estará disponível no seu navegador.

3Acesso ao Dashboard

Por padrão, o dashboard estará disponível em:

http://127.0.0.1:8050/

Clique no botão abaixo para tentar abrir o dashboard (supondo que o servidor já esteja em execução):

4Funcionalidades

  • Visualização 3D das soluções não-dominadas
  • Explorador de soluções individuais com detalhes de cada roteiro
  • Gráficos da evolução do algoritmo ao longo das gerações
  • Timeline detalhado de cada solução com tempos e modos de transporte
